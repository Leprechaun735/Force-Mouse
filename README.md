# Force-Mouse
Opening and closing computer programs with hand gestures

Using OpenCV, pyautogui, and mediapipe to track my hand and determine how many fingers are up at a time letting me send up to 32 different imputs tyo do whatever I need it to do.

For example holding up just my pinky finger will open a new window while flipping the computer off will actually shut it off.
